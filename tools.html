🛠️ قائمة أدوات SaToor X7 (60 أداة)

1- أداة DDOS هجوم وهمي

أداة لتنفيذ هجمات DDOS تعليمية على الروابط أو المواقع لتجربة الضغط فقط.

أمر التثبيت: 
git clone https://github.com/Satoorx7/ddos-tool
طريقة التشغيل: 
cd ddos-tool && python3 ddos.py
🎥 شرح فيديو

2- أداة تحميل فيديوهات بدون علامة مائية

تحميل فيديوهات من TikTok وYouTube وInstagram بجودات مختلفة بدون علامات مائية.

أمر التثبيت: 
git clone https://github.com/Satoorx7/video-downloader
طريقة التشغيل: 
cd video-downloader && python3 main.py
🎥 شرح فيديو

3- أداة تثبيت Termux للمبتدئين

تقوم بتثبيت الأدوات الأساسية تلقائيًا للمستخدمين الجدد على Termux.

أمر التثبيت: 
bash <(curl -s https://raw.githubusercontent.com/Satoorx7/auto-install/main/setup.sh)
طريقة التشغيل: 
اتبع التعليمات بعد تشغيل السكربت
🎥 شرح فيديو
